Extension
=========

Description
-----------
Search for youtube videos using the text selected


Name
----
Youtube Search
YouSearch
MusicSearch

User Stories
------------
1. Search Youtube for selected text using browser action	
2. Play Youtube video inline the search results				
3. Open Youtube video in new tab from search results		
4. Provide link to iTunes Store from search results			
5. Search Youtube from popup window							
6. Begin Youtube search from popup window

2. Save playlist to user account
